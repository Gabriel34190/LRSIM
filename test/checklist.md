# Checklist de tests manuels

## Responsive
- [ ] Accueil: affichage correct sur mobile (≤ 375px), tablette (~768px), desktop (≥ 1280px)
- [ ] Navbar: menu/CTA visibles et utilisables sur toutes tailles
- [ ] Détail appartement: galerie, specs et carte lisibles sans débordement

## Galerie / Images
- [ ] Upload image appartement (admin): aperçu immédiat visible
- [ ] Suppression image: vignette retirée et image principale mise à jour
- [ ] DPE réduit/détaillé: import PNG/JPG et aperçu modal fonctionnel

## Carte
- [ ] Localisation calculée depuis l'adresse (marker visible)
- [ ] Popup marker affiche nom + adresse + lien itinéraire
- [ ] Edition d'adresse (admin): sauvegarde, normalisation, et recentrage de la carte

## Formulaire de contact
- [ ] Validation email (format invalide bloqué)
- [ ] Génération du mailto avec objet et corps préremplis

## Disponibilités
- [ ] Changement “Disponible/Indisponible” (admin) met à jour l’affichage
- [ ] Saisie de date fin de contrat visible en indisponibilité

## Accessibilité rapide
- [ ] Focus visible sur éléments interactifs (boutons, liens)
- [ ] Navigation clavier possible sur boutons principaux
